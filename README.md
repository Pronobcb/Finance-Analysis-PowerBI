# 💰Finance-Analysis-PowerBI
📌 Overview
FinSight is an interactive Power BI Finance Dashboard designed to analyze financial transactions, customer behavior, and revenue performance. The dashboard provides a comprehensive view of key financial metrics, helping businesses track growth, monitor transaction trends, and identify high-value customer segments.

## 📊 Dashboard Highlights
- Key Performance Indicators (KPIs)
- Total Amount: 455.53M
- Total Transactions: 50.00K
- Average Transaction Value: 9.11K
- Total Fees: 726.24K
- Total Tax: 130.74K
- Interactive Analysis

- ✔ Year-wise filtering
- ✔ Occupation-wise analysis
- ✔ Category-wise analysis
- ✔ Dynamic metric selection
- ✔ Drill-down financial insights

## 📈 Visualizations Included
# 1. Monthly Financial Trend

- Tracks transaction amounts across different months to identify seasonal patterns and business performance trends.

# 2. Transaction Status Analysis

Breaks down transaction amounts by:
- Success
- Failed
- Pending
# 3. Customer Segment Analysis
Analyzes revenue contribution from:
- Retail
- Premium
- SME
- Corporate
- Wealth
# 4. State-wise Performance

Compares transaction amounts across different states to identify top-performing regions.

# 5. Gender-wise Analysis

Examines financial contribution and transaction behavior by gender.

#6.Transaction Type Breakdown

Provides detailed insights into:

- Amount
- Tax
- Fees
- Number of Transactions

for each transaction category.
